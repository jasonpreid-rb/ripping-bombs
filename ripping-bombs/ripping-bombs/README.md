# Ripping Bombs 🏌️
### Global Longest Golf Drive Database

## Quick Start

```bash
npm install
npm run dev
```
Open http://localhost:5173

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → Add New Project → Import repo
3. Vercel auto-detects Vite — click Deploy
